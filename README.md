### 🎫 [EventQR Pro](https://github.com/AKVishwakrama/QR_Project)
**Enterprise Event Management System** • *Dec 2024*

Production-grade ticketing platform with QR validation, live analytics, and AWS cloud deployment.

```
Backend: Java REST API • PostgreSQL • 8 RESTful endpoints
Frontend: Vanilla JS • QRCode.js • Responsive Design
Cloud: AWS (EC2, RDS, S3, CloudFront) with HTTPS
```

**Architecture Highlights:**
- 🔒 SHA-256 encrypted QR codes with duplicate prevention
- 📊 Real-time analytics dashboard (revenue, attendance, trends)
- ⚡ PreparedStatements for SQL injection prevention
- 🌍 Three-tier architecture: S3 + EC2 + RDS
